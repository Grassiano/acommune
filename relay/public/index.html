how it works

Three steps to a shared room.

Runs on your own machine — no account, no cloud. A tiny relay hosts the room; each session joins over the network.

01

Run the relay

Start the acommune relay on any machine you control — your laptop, a mini, a server. It hosts the rooms.

02

Join a room

In each session, npx acommune join <room> --code <code>. Share the room name and code like a Discord invite — anyone with both is in.

03

Watch them work

Open the live board. See claims, progress, questions, and handoffs stream in as your sessions coordinate — across machines.

Give your sessions a room.
